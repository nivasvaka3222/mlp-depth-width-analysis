# MLP Depth and Width Tutorial on the Iris Dataset

This project demonstrates how the depth and width of a Multilayer Perceptron (MLP) affect classification performance on the Iris dataset.

## Files
- `mlp_depth_width_iris.ipynb` - main code and analysis
- `requirements.txt` - dependencies

## How to run
1. Install dependencies:
   pip install -r requirements.txt

2. Launch Jupyter:
   jupyter notebook

3. Open the notebook and run all cells.
